# Project Summary

- Introduced a metadata-driven adapter architecture aligned with official docs references for `claude`, `codex`, `qwen`, and `antigravity`.
- Upgraded canonical standards to include `skills`, `workflows`, `mcp`, and `agents` catalogs with richer production-oriented prompt content.
- Added schema validation gate before `init/link/sync/update` and improved backup handling for symlink-based installs.
- Enhanced CLI UX with branded banner, status icons, verbose/json doctor modes, and clearer diagnostics.
- Added keyboard-driven interactive wizard mode for no-argument launch with arrow/enter navigation and space-based agent multi-select, plus TTY fallback behavior for non-interactive environments.
- Expanded compatibility matrix to include docs provenance, verification state, and asset-kind coverage.
- Published bilingual documentation: `README.md` (English) and `README.vi.md` (Vietnamese).
